### 1. E-commerce Customer Behaviour Analysis
**Tools Used:** Python, SQL, Power BI

- **Key Deliverables:** - Developed an RFM (Recency, Frequency, Monetary) model to segment users.

### 2. Road Accident Data Analysis & Dashboard
**Tools Used:** MySQL, Power Query, Power BI
- **Objective:** Investigated UK Road Safety data (2021-2022) to pinpoint accident hotspots and contributing factors.
- **Key Deliverables:**
  - Cleaned and modeled 50,000+ rows of raw accident data.
  - Built a dynamic dashboard featuring KPIs for casualties by vehicle type and weather conditions.
  - **Impact:** Provided insights into peak accident hours, suggesting high-priority times for emergency service deployment.
